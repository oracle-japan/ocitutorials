    Oracle Data Safe チュートリアルまとめ

    Oracle Data Safeは、Oracle CloudのDatabase Serviceをよりセキュアに使うための付加的なサービスで、機密データの発見、(データ・マスキング、アクティビティ監査、セキュリティ構成の評価、ユーザーのリスク評価などを、GUIツールから簡単に実施できます。このまとめページでは、Data Safeを有効化から、各種の使い方について説明しています。

    • アイデンティティとセキュリティ編

      • エッジポリシーのWeb Application Firewallのログを分析する

      • OCI Load Balancerに直接アタッチするタイプのWeb Application Firewallのログを分析する

      • OCI Load Balancerに直接アタッチするタイプのWeb Application Firewallを構成する

      • OCI Network Firewallを構築する

      • OCI Web Application Firewallのエッジポリシーを使ってWebサーバを保護する

      • OCI Network FirewallのIDS/IPS機能を検証する

      • OCI Network Firewallのログを分析する

      • Vaultサービスを使ってBase Databaseをユーザー管理の暗号鍵で暗号化する

      • Vaultサービスを使ってコンピュート・インスタンスのブート・ボリュームをユーザー管理の暗号鍵で暗号化する

      • Vaultサービスを使ってObject Storageをユーザー管理の暗号鍵で暗号化する

      • Vaultを作成し 顧客管理の鍵をインポートする

      • Web Application Firewall(WAF)を使ってWebサーバを保護する

      • OCI IAM Identity DomainsとAzure ADとの認証連携（外部IDP連携）を設定する

      • OCI IAM Identity Domainsのドメインの追加とライセンスタイプを変更する

      • OCI IAM Identity Domains - テナント管理者・一般ユーザーを作成する

      • Cloud Guardを使ってみる

      • Oracle Data Safe チュートリアルまとめ

      • OCI IAM Identity DomainでAPEXで作成したアプリケーションに認証と認可をする

      • OCI IAM Identity DomainでユーザーのMFAを有効化する

      • Security Zoneを有効化する

      • Vulnerability Scanning Serviceを設定する
